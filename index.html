Команды Git
Команда Описание
git init Команда git init инициализирует новый локальный репозиторий Git, превращая текущую рабочую директорию в отслеживаемый проект
git clone url -
git add file Команда git clone
git add . git add — это команда Git, которая добавляет содержимое изменённых или новых файлов в «индекс»
git commit -m "comment" git commit файлы создаёт новый коммит, в основе которого лежат уже существующие файлы, добавляя изменения только для указанных файлов
git commit -a -m "comment" git commit - редактирует репозиторий, заносит туда изменения
git push url Команда Git push позволяет отправлять локальную ветку на удаленный репозиторий.
git status git status — это команда Git, которая показывает текущее состояние рабочего каталога и индекса
git log Команда git log используется для просмотра истории коммитов в репозитории
git branch branch_name -
git checkout branch_name -
git checkout -b branch_name Команда git checkout branch_name — это основной инструмент в Git для переключения между существующими ветками проекта
git remote Команда git remote в Git используется для управления подключениями к удаленным репозиториям
